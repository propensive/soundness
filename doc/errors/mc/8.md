# `SN-mc/8` — Type has multiple `flatMap` methods

Raised at compile time by Mercator's `monad` macro when the target
type has more than one `flatMap` overload.

## Cause

Multiple `flatMap` overloads are ambiguous for derivation purposes.

## Example message

```
[↯SN-mc/8] the type Foo has more than one possible flatMap method
```

## Resolution

Remove or rename overloads, or supply a `Monad` instance manually.
