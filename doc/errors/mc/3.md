# `SN-mc/3` — Missing `Identity` for functor derivation

Raised at compile time by Mercator's `functor` macro when no
`Identity` instance can be summoned for the type.

## Cause

Mercator's `Functor` derivation depends on an underlying `Identity`
instance.

## Example message

```
[↯SN-mc/3] could not find Identity value for List
```

## Resolution

Bring an `Identity` instance into scope (or provide one), then
re-derive the `Functor`.
