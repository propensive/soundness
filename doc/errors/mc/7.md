# `SN-mc/7` — Type has no `flatMap` method

Raised at compile time by Mercator's `monad` macro when the target
type has no `flatMap` method.

## Cause

Mercator's `Monad` derivation requires a `flatMap` on the type.

## Example message

```
[↯SN-mc/7] the type Foo has no flatMap methods
```

## Resolution

Define a `flatMap` on the type, or supply a `Monad` instance
manually.
