# `SN-mc/6` — Missing `Functor` for monad derivation

Raised at compile time by Mercator's `monad` macro when no
`Functor` instance can be summoned for the type.

## Cause

Mercator's `Monad` derivation depends on an underlying `Functor`
instance.

## Example message

```
[↯SN-mc/6] could not find Functor value for Foo
```

## Resolution

Bring or derive a `Functor` instance first.
