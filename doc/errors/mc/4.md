# `SN-mc/4` тАФ Type has no `map` method

Raised at compile time by Mercator's `functor` macro when the
target type has no `map` method.

## Cause

Mercator's `Functor` derivation requires a `map` method on the
type.

## Example message

```
[тЖпSN-mc/4] the type Foo has no map methods
```

## Resolution

Define a `map` method on the type, or supply a `Functor` instance
manually.
