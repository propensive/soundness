# `SN-pf/2` — `TerminalError`

Raised when interactive terminal operations are attempted but
stdin is not connected to a TTY.

## Cause

Profanity needs a real terminal for interactive features. When
stdin is a pipe or redirect, this error is thrown.

## Example message

```
[↯SN-pf/2] STDIN is not attached to a TTY
```

## Resolution

Run the program in an interactive shell, or fall back to a
non-interactive code path.
