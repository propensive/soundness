# `SN-pf/1` тАФ `DismissError`

Raised when an interactive prompt is dismissed by the user.

## Cause

Profanity throws this error when the user presses an
interaction-cancelling key (e.g. `Esc`) on a prompt that requires a
selection.

## Example message

```
[тЖпSN-pf/1] the user dismissed an interaction
```

## Resolution

Treat the dismissal as a graceful cancellation in the calling
code.
