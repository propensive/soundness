# `SN-hp/5` — Hex literal contains non-hex character

Raised at compile time by Hypotenuse's `hex"…"` literal macro when
the body contains a character that is not a hexadecimal digit.

## Cause

Hex literals admit only `0`-`9`, `a`-`f`, and `A`-`F`.

## Example message

```
[↯SN-hp/5] xy is not a valid hexadecimal character
```

## Resolution

Remove or correct the offending character.
