# `SN-hp/14` тАФ `S8` value above `Byte.MaxValue`

Raised at compile time by Hypotenuse's `s8` macro when the value
exceeds `127`.

## Cause

An `S8` (signed 8-bit) is bounded above by `Byte.MaxValue`.

## Example message

```
[тЖпSN-hp/14] an S8 may not be greater than 127
```

## Resolution

Provide a value in the range `-128`тАУ`127`.
