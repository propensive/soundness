# `SN-hp/8` ŌĆö `U16` value above 65535

Raised at compile time by Hypotenuse's `u16` macro when the value
is greater than 65535.

## Cause

A `U16` (unsigned 16-bit) is bounded above by `0xffff`.

## Example message

```
[Ōå»SN-hp/8] a U16 may not be greater than 65535
```

## Resolution

Provide a value in the range 0-65535.
