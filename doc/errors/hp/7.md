# `SN-hp/7` ŌĆö `U16` value below 0

Raised at compile time by Hypotenuse's `u16` macro when the value
is less than 0.

## Cause

A `U16` (unsigned 16-bit) cannot be negative.

## Example message

```
[Ōå»SN-hp/7] a U16 may not be less than 0
```

## Resolution

Provide a value in the range 0-65535.
