# `SN-hp/13` тАФ `S8` value below `Byte.MinValue`

Raised at compile time by Hypotenuse's `s8` macro when the value
is below `-128`.

## Cause

An `S8` (signed 8-bit) cannot represent values below `Byte.MinValue`.

## Example message

```
[тЖпSN-hp/13] an S8 may not be less than -128
```

## Resolution

Provide a value in the range `-128`тАУ`127`.
