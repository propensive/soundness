# `SN-hp/9` тАФ `S16` value below `Short.MinValue`

Raised at compile time by Hypotenuse's `s16` macro when the value
is below `-32768`.

## Cause

An `S16` (signed 16-bit) cannot represent values below `Short.MinValue`.

## Example message

```
[тЖпSN-hp/9] an S16 may not be less than -32768
```

## Resolution

Provide a value in the range `-32768`тАУ`32767`.
