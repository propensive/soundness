# `SN-hp/4` — Binary literal must be 8, 16, 32, or 64 bits

Raised at compile time by Hypotenuse's `bin"…"` literal macro when
the literal's bit count is not one of the supported widths.

## Cause

Binary literals are typed by their bit count; only `8`, `16`,
`32`, and `64` are supported.

## Example message

```
[↯SN-hp/4] a binary literal must be 8, 16, 32 or 64 bits long
```

## Resolution

Pad or trim the literal to one of the supported widths.
