# `SN-hp/3` — Binary literal contains non-binary character

Raised at compile time by Hypotenuse's `bin"…"` literal macro when
the body contains a character that is not `0` or `1`.

## Cause

Binary literals must be exclusively `0` and `1`.

## Example message

```
[↯SN-hp/3] a binary value can only contain characters '0' or '1'
```

## Resolution

Remove or correct the offending character.
