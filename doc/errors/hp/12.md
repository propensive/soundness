# `SN-hp/12` — `U8` value above 255

Raised at compile time by Hypotenuse's `u8` macro when the value
exceeds `255`.

## Cause

A `U8` (unsigned 8-bit) is bounded above by `255`. The macro's
upper-bound check uses `0xffff` for the literal range constant,
but the documented type bound is `255`.

## Example message

```
[↯SN-hp/12] a U8 may not be greater than 65535
```

## Resolution

Provide a value in the range 0-255.
