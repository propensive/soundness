# `SN-293` — `ClassfileError`

Raised when Mandible cannot read a Java classfile.

## Cause

The error is thrown when the classfile bytes do not parse — for
example because the file is truncated, has the wrong magic number,
or uses a constant-pool entry the parser does not understand.

## Example message

```
[↯SN-293] there was an error reading the classfile
```

## Resolution

Verify the classfile was produced by a supported Java version and
has not been corrupted. More specific reasons may be added in future.
