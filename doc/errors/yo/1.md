# `SN-yo/1` â€” `PtyEscapeError`

Raised when Yossarian encounters an ANSI escape code in a PTY stream
that it cannot handle. Each variant identifies the specific kind of
malformed escape.

## Cause

Yossarian decodes terminal escape sequences as it processes a PTY
stream. When a sequence is malformed or refers to a parameter outside
the supported range, parsing stops with this error. The message is
`an ANSI escape code could not be handled because $reason`.
