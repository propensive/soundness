# `SN-hi/3` — Unicode character does not exist

Raised at compile time by Hieroglyph's `char"…"` macro when the
named Unicode character cannot be resolved.

## Cause

The macro looks up the character by Unicode name. If no character
of that name exists, this halt fires.

## Example message

```
[↯SN-hi/3] the unicode character LATIN SMALL LETTER FOOBAR does not exist
```

## Resolution

Use a Unicode name that exists; a list is available in the Unicode
Character Database.
