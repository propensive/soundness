# `SN-hi/1` — `CharEncodeError`

Raised when Hieroglyph cannot encode a character with the chosen
encoding.

## Cause

Hieroglyph throws this error when encoding a character that the
target encoding cannot represent (e.g. a non-Latin character with
ISO-8859-1).

## Example message

```
[↯SN-hi/1] character ñ cannot be encoded with the encoding US-ASCII
```

## Resolution

Either choose an encoding that supports the character (UTF-8 is a
safe default) or pre-process the input to remove unrepresentable
characters.
