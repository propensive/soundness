# `SN-fu/3` — Realm code must be lowercase letters

Raised at compile time by Fulminate's `realm"…"` macro when the
provided code contains anything other than lowercase ASCII letters.

## Cause

The realm code is a two-letter module identifier (e.g. `realm"ur"`)
used to render `SN-xx/d.e` codes in error messages. The macro
rejects any other characters.

## Example message

```
[↯SN-fu/3] the realm code should contain only lowercase letters
```

## Resolution

Use only the lowercase letters `a-z` in the realm code.
