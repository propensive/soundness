# `SN-fu/2` — `UncheckedError`

Raised when a non-Soundness `Throwable` must be wrapped as an
`Error`.

## Cause

`Error.apply(throwable)` either passes through an existing `Error`
or wraps a foreign throwable in `UncheckedError`. The wrapper
preserves the original message (or notes its absence) and the
stack trace from the original throwable.

## Example messages

```
[↯SN-fu/2] unchecked error with message, NullPointerException
```

```
[↯SN-fu/2] unchecked error with no message
```

## Resolution

Identify the underlying `Throwable` (`getCause` chain) and decide
whether to surface it as a typed Soundness error or fix the source
to avoid the foreign throw.
