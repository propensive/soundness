# `SN-le/1` — `QueryError`

Raised when Legerdemain cannot read a query parameter.

## Cause

Query-parameter decoding fails when the textual value cannot be
converted to the expected type.

## Example message

```
[↯SN-le/1] the query parameter could not be read
```

## Resolution

Confirm the query parameter is present and has the expected
format; supply a default if appropriate. FIXME: more specific
reasons may be added.
