# `SN-le/3` ‚Äî Missing `Parametric` value

Raised at compile time by Legerdemain's `query` macro when no
contextual `Parametric` instance is available for the inferred key
type.

## Cause

The macro requires a `Parametric` typeclass instance to encode the
key into a query string. If none is in scope, the macro halts.

## Example message

```
[‚ÜØSN-le/3] could not find a contextual Parametric value for foo
```

## Resolution

Bring an appropriate `given Parametric` into scope, or change the
key type to one that has a `Parametric` instance.
