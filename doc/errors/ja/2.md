# `SN-ja/2` — `JsonPointerError`

Raised when a JSON pointer cannot be resolved against a value.

## Cause

Jacinta resolves JSON pointers per RFC 6901. This error is thrown
when a segment of the pointer does not name a key in the current
object or a valid index in the current array.

## Example message

```
[↯SN-ja/2] could not resolve JSON pointer
```

## Resolution

Verify the pointer matches the structure of the document; check
escapes (`~0` for `~`, `~1` for `/`).
