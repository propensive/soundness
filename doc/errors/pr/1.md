# `SN-pr/1` — `RetryError`

Raised when Parasite gives up retrying an operation after the
configured number of attempts.

## Cause

Parasite retries an operation up to `count` times. If every attempt
fails, this error is thrown.

## Example message

```
[↯SN-pr/1] aborted repeated evaluation after 5 attempts
```

## Resolution

Inspect logs to identify why every attempt failed; either fix the
underlying problem or increase the attempt limit if transient
failure is expected.
