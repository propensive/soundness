# `SN-he/1` — `ClasspathError`

Raised when Hellenism cannot find a resource on the classpath.

## Cause

Hellenism resolves resources by name; this error is thrown when no
classpath entry holds the requested resource.

## Example message

```
[↯SN-he/1] the resource config.json was not on the classpath
```

## Resolution

Confirm the resource path (relative to the classpath root) and
that the resource is included in the build artefact.
