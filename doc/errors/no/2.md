# `SN-no/2` — Missing `Nominative` instance

Raised at compile time by Nomenclature's `parse` macro when no
`Nominative` instance is in scope for the requested system.

## Cause

The macro expands name parsing against a system specified by type;
without a `given Nominative`, it cannot proceed.

## Example message

```
[↯SN-no/2] Couldn't find a `Nominative` instance on system
```

## Resolution

Bring a `given Nominative` for the system into scope, or provide
one.
