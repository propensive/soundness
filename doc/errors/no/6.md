# `SN-no/6` — Constraint inaccessible

Raised at compile time by Nomenclature's `parse` macro when the
type-level constraint cannot be accessed.

## Cause

FIXME: the message is intentionally generic. The macro could not
read the constraint expression at compile time.

## Example message

```
[↯SN-no/6] Could not access constraint
```

## Resolution

FIXME: typically indicates an internal misconfiguration; ensure
the constraint type is accessible from the call site.
