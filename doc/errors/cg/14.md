# `SN-cg/14` — `mitigate` argument must be a partial function

Raised when `mitigate` is invoked with an argument that is not a
match-expression-style partial function.

## Cause

`mitigate` requires that its argument be a partial function written
as `case … => …` clauses, so that the macro can analyse each branch
to determine which errors are recovered.

## Example message

```
[↯SN-cg/14] argument to `mitigate` should be a partial function implemented as match cases
```

## Resolution

Rewrite the argument as `{ case … => …; case … => … }`.
