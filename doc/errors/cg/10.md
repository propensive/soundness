# `SN-cg/10` — Wildcard in pattern type extraction

Raised by Contingency's pattern-type extractor when a wildcard
pattern is encountered where a concrete type was needed.

## Cause

A wildcard `_` pattern carries no type information that the
extractor can use to register a recovered case.

## Example message

```
[↯SN-cg/10] wildcard
```

## Resolution

Replace the wildcard with a typed pattern (e.g. `error: SomeError`).
