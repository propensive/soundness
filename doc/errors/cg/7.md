# `SN-cg/7` — Pattern not exhaustive

Raised by Contingency's pattern-analysis macro when a partial
function does not cover every possible error type.

## Cause

`mitigate`/`recover` patterns must cover every error variant that
can flow into them. When a case is missing, this halt fires.

## Example message

```
[↯SN-cg/7] the pattern is not exhaustive
```

## Resolution

Add the missing case(s), or use a wildcard fallback.
