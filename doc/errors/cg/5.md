# `SN-cg/5` — Unexpected AST node

Raised by Contingency's pattern-analysis macro when it encounters a
tree shape it does not handle.

## Cause

The macro walks user-supplied patterns to determine error
exhaustiveness. Some AST shapes are not anticipated by the analysis
and produce this halt with the offending node's `toString`.

## Example message

```
[↯SN-cg/5] unexpected AST: <node>
```

## Resolution

Rewrite the pattern in a simpler form. FIXME: if the AST is from
ordinary Scala source, please file an issue with the pattern that
triggered this halt.
