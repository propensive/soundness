# `SN-415` — `JsonPointerError`

Raised when a JSON pointer cannot be resolved.

## Cause

Jacinta resolves JSON pointers per RFC 6901. The constructor takes
a `reason: Reason`; the rendered message is `the JSON pointer
could not be resolved because $reason`.

## Variants

### `SN-415.1` — `UnknownDocument`

The pointer's URL was not registered with the
`JsonPointer.Registry`, so the document it refers to could not be
located.

**Example.** `[↯SN-415.1] the JSON pointer could not be resolved because the registry contains no document at the pointer's URL`

**Resolution.** Register the document with `registry(url) =
document` before resolving pointers that reference it; or
override `Registry.lookup` to fetch the document on demand.
