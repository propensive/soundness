# `SN-an/1` — `DataError`

Raised when Anamnesis encounters a database error.

## Cause

Anamnesis wraps the underlying database driver. Any error reported
by the driver is surfaced as `DataError`. The message itself is
generic; consult the cause chain for specifics.

## Example message

```
[↯SN-an/1] there was a database error
```

## Resolution

Inspect the cause chain (or driver logs) for the underlying SQL or
connection error. FIXME: more specific variants may be added as the
module evolves.
