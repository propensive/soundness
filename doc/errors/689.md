# `SN-689` ŌĆö `U8` value below 0

Raised at compile time by Hypotenuse's `u8` macro when the value is
less than 0.

## Cause

A `U8` (unsigned 8-bit) cannot be negative.

## Example message

```
[Ōå»SN-689] a U8 may not be less than 0
```

## Resolution

Provide a value in the range 0-255.
