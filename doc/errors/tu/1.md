# `SN-tu/1` — `StreamError`

Raised when Turbulence's stream is cut short before the expected
number of bytes have been read.

## Cause

The error is thrown when an upstream source closes mid-stream (or
hits EOF earlier than expected). The constructor records the
`total` byte count read so far.

## Example message

```
[↯SN-tu/1] the stream was cut prematurely after 1024
```

## Resolution

Investigate why the source ended early — typical causes are
network errors, truncated files, or cancelled producers.
