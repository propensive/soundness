# `SN-955` — Time literal: negative value

Raised by Aviation's `validTime` macro when a time literal is
negative.

## Cause

A clock face cannot represent negative hours.

## Example message

```
[↯SN-955] a time cannot be negative
```

## Resolution

Use a non-negative literal value.
