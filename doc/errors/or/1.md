# `SN-or/1` — `OAuthError`

Raised when an OAuth flow against a remote provider fails.

## Cause

Orthodoxy mediates OAuth 1.0/2.0 exchanges. Any failure — network,
parsing, status, or scope — surfaces as one of the variants below.
The message is `OAuth failed because $reason`.

## Variants

### `SN-or/1.1` — `Connection`

A TCP/TLS connection to the OAuth provider failed.

**Example.** `[↯SN-or/1.1] OAuth failed because could not connect to the OAuth provider at https://example.com because the connection was refused`

**Resolution.** Verify network connectivity, the provider URL, and
TLS configuration. Inspect the inner `ConnectError.Reason` for
specifics.

### `SN-or/1.2` — `InvalidJsonResponse`

The provider returned a body that could not be parsed as JSON.

**Example.** `[↯SN-or/1.2] OAuth failed because Invalid JSON response`

**Resolution.** Check the provider's documentation for the expected
response format; inspect the raw response for clues.

### `SN-or/1.3` — `UnexpectedHttpStatus`

The provider responded with a status outside the expected set.

**Example.** `[↯SN-or/1.3] OAuth failed because the provider returne an unexpected HTTP status: 500`

**Resolution.** Consult the provider's status documentation; many
providers return 4xx for client errors with diagnostic JSON in the
body.

### `SN-or/1.4` — `InsufficientPrivileges`

The user did not grant a scope that the application requested.

**Example.** `[↯SN-or/1.4] OAuth failed because the user has not granted access to read:user`

**Resolution.** Re-prompt the user including the required scope, or
gracefully degrade functionality that needs it.

### `SN-or/1.5` — `Unauthorized`

The user explicitly denied authorisation, or the provider rejected
the credentials.

**Example.** `[↯SN-or/1.5] OAuth failed because authorization was not granted`

**Resolution.** Surface a clear message to the user and re-initiate
the OAuth flow if appropriate.

### `SN-or/1.6` — `Other`

A miscellaneous error did not fit the categories above.

**Example.** `[↯SN-or/1.6] OAuth failed because an unexpected error occurred`

**Resolution.** Inspect logs and any cause attached to the error;
FIXME: variant is intentionally generic — a more specific reason may
be added in the future.
