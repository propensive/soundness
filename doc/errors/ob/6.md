# `SN-ob/6` тАФ Missing `Online` instance

Raised at compile time by Obligatory's `rpc` macro when no
contextual `Online` instance is available.

## Cause

The macro requires an `Online` instance to confirm network
operations are permitted in the current scope.

## Example message

```
[тЖпSN-ob/6] a contextual `Online` instance is required
```

## Resolution

Bring an `Online` instance into scope at the call site.
