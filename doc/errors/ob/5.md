# `SN-ob/5` — Missing `Tactic[JsonRpcError]`

Raised at compile time by Obligatory's `rpc` macro when no
contextual `Tactic[JsonRpcError]` instance is available.

## Cause

The macro requires a `Tactic` to know how to surface JSON-RPC
errors at the call site.

## Example message

```
[↯SN-ob/5] could not find a contextual `Tactic[JsonRpcError]` instance
```

## Resolution

Bring a `Tactic[JsonRpcError]` into scope (typically via
`mitigate`/`unfold`).
