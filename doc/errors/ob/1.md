# `SN-ob/1` ŌĆö `SseError`

Raised when Obligatory cannot parse a server-sent event.

## Cause

SSE messages are framed as `field: value` lines. When a frame
cannot be parsed against this format, this error is thrown.

## Example message

```
[Ōå»SN-ob/1] the server-sent event was not valid
```
