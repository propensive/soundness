# `SN-ob/4` — `JsonRpcError`

Raised when a JSON-RPC operation fails.

## Cause

Obligatory's JSON-RPC dispatcher raises this error when a request
cannot be processed. The constructor takes a `reason: Reason`; the
rendered message is `the JSON-RPC operation failed because
$reason`.

## Variants

### `SN-ob/4.1` — `UnknownMethod`

The `method` field of the incoming JSON-RPC request did not match
any of the methods declared on the dispatcher's interface — the
wildcard fallback in the generated dispatcher.

**Example.** `[↯SN-ob/4.1] the JSON-RPC operation failed because the method name was not recognised by the dispatcher`

**Resolution.** Confirm the method name matches a method on the
RPC interface; check for typos.
