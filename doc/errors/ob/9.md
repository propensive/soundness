# `SN-ob/9` — RPC method must have one parameter list

Raised at compile time by Obligatory's `rpc` macro when a method
signature has more than one parameter list (or none).

## Cause

JSON-RPC methods are mapped to Scala methods with exactly one
parameter list — the one carrying the request fields.

## Example message

```
[↯SN-ob/9] the method foo must have exactly one parameter list
```

## Resolution

Restructure the method to take exactly one parameter list.
