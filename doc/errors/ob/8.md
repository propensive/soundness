# `SN-ob/8` — Missing `Monitor` instance

Raised at compile time by Obligatory's `rpc` macro when no
contextual `Monitor` instance is available.

## Cause

The macro requires a `Monitor` to schedule and supervise async
work.

## Example message

```
[↯SN-ob/8] a contextual `Monitor` instance is required
```

## Resolution

Bring a `Monitor` into scope at the call site (e.g. via `Daemon`).
