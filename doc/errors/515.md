# `SN-515` — `IsinError`

Raised when a value cannot be parsed as a valid ISIN (International
Securities Identification Number).

## Cause

Plutocrat checks that an ISIN is twelve characters long, starts with a
recognised two-letter country code, contains only digits and uppercase
letters, and passes the Luhn checksum. The message is `the ISIN
number is not valid because $reason`.

## Variants

### `SN-515.1` — `InvalidCharacter`

A character is not a digit or an uppercase letter.

**Example.** `[↯SN-515.1] the ISIN number is not valid because the character - at position 4 is not a digit or uppercase letter`

**Resolution.** Use only `0`–`9` and `A`–`Z`.

### `SN-515.2` — `BadCountryCode`

The first two characters are not a recognised ISO 3166-1 alpha-2
country code.

**Example.** `[↯SN-515.2] the ISIN number is not valid because its country code XX was not valid`

**Resolution.** Use a current ISO country code as the prefix.

### `SN-515.3` — `WrongLength`

The total length is not 12 characters.

**Example.** `[↯SN-515.3] the ISIN number is not valid because it had length 11, but it should be 12 characters long`

**Resolution.** Pad or trim the input to exactly 12 characters of the
required form.

### `SN-515.4` — `LuhnCheck`

The final check digit does not match the Luhn checksum of the
preceding 11 characters.

**Example.** `[↯SN-515.4] the ISIN number is not valid because its last digit failed the Luhn check`

**Resolution.** Recompute the check digit, or correct the typo in the
ISIN body that caused the mismatch.
