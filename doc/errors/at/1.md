# `SN-at/1` — `CompilerError`

Raised when Anthology cannot honour a compiler-configuration
request.

## Cause

Anthology programmatically configures and runs the Scala compiler.
This error reports a problem with the configuration — for example
missing classpath entries, an invalid option, or a target that the
embedded compiler does not support.

## Example message

```
[↯SN-at/1] there was a problem with the compiler configuration
```
