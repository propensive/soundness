# `SN-ab/1` — Quanta does not include this unit

Raised when the `Quanta` macro is asked to look up a unit that is not part
of the `Quanta` it was applied to.

## Cause

The Abacist macros derive a multiplier for a unit by searching the
`Quanta` type for that unit's power. If the requested unit is not present
in the `Quanta`, the search fails and the macro aborts.

## Example message

```
[↯SN-ab/1] the Quanta does not include this unit
```

## Resolution

Either include the unit in the `Quanta` type at the call site, or call
the lookup with a unit that is already part of the `Quanta`.
