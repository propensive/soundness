# `SN-dg/1` — `FqcnError`

Raised when a value is not a valid fully-qualified class name.

## Cause

Digression validates FQCNs as a sequence of dot-separated identifier
segments, each conforming to Java identifier rules. The message is
`the class name $name is not valid because $reason`.

## Variants

### `SN-dg/1.1` — `InvalidChar`

A character in a name segment is not legal in a Java identifier.

**Example.** `[↯SN-dg/1.1] the class name foo.b-r.Baz is not valid because a package name may not contain the character -`

**Resolution.** Remove or replace the offending character.

### `SN-dg/1.2` — `InvalidStart`

A name segment starts with a character not legal as the first
character of an identifier (typically a digit).

**Example.** `[↯SN-dg/1.2] the class name foo.1bar.Baz is not valid because a package name may not start with the character 1`

**Resolution.** Begin each segment with a letter, `_`, or `$`.

### `SN-dg/1.3` — `EmptyName`

A name segment is empty — typically caused by `..` in the input or a
leading or trailing dot.

**Example.** `[↯SN-dg/1.3] the class name foo..Bar is not valid because a package name cannot be empty`

**Resolution.** Remove the empty segment.

### `SN-dg/1.4` — `JavaKeyword`

A name segment is a reserved Java keyword.

**Example.** `[↯SN-dg/1.4] the class name foo.class.Baz is not valid because a package name cannot be the Java keyword, class`

**Resolution.** Choose a non-keyword segment.
