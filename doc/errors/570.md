# `SN-570` тАФ `AuthError`

Raised when an HTTP authentication value is not valid.

## Cause

Telekinesis throws this error when an `Authorization` header value
(or similar) does not match the expected scheme syntax.

## Example message

```
[тЖпSN-570] the authentication value Basic ??? is not valid
```

## Resolution

Verify the value matches the expected scheme (`Basic`, `Bearer`,
etc.) and is correctly encoded.
