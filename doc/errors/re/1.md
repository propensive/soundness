# `SN-re/1` — `SemverError`

Raised when a value is not a valid semantic version.

## Cause

Revolution validates inputs against the SemVer 2.0 grammar:
`MAJOR.MINOR.PATCH[-prerelease][+build]`. Anything else is
rejected.

## Example message

```
[↯SN-re/1] 1.2 is not a valid semantic version
```

## Resolution

Provide a SemVer-compliant version string, or pad missing
components.
