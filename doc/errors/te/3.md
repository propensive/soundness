# `SN-te/3` — `HttpError`

Raised when an HTTP request received an error-class status from
the server.

## Cause

Telekinesis throws this error when the response status is not in
the success range. The constructor captures the `Http.Status` and
the response `headers` for inspection.

## Example message

```
[↯SN-te/3] the HTTP request failed with status 500
```

## Resolution

Inspect the status and headers to determine the appropriate
recovery; for client errors (4xx) the request itself is at fault,
while server errors (5xx) usually warrant a retry.
