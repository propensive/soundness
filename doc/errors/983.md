# `SN-983` — `RpcError`

Raised when an RPC call fails for a reason that cannot be
expressed by the JSON-RPC error type.

## Example message

```
[↯SN-983] the remote procedure call failed
```

## Resolution

Inspect the cause chain for transport- or protocol-level details.
