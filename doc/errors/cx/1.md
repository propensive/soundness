# `SN-cx/1` — `BindError`

Raised when Coaxial cannot bind a socket to the requested port.

## Cause

The OS refused to allocate the requested port — typically because
another process is already bound to it, or because the port is
privileged and the process lacks permission.

## Example message

```
[↯SN-cx/1] the port was not available for binding
```

## Resolution

Free the port (stop the conflicting process), choose a different
port, or run with sufficient privilege if the port requires it
(< 1024 on POSIX).
