# `SN-ta/1` ‚Äî `WebDriverError`

Raised when Tarantula's WebDriver action fails inside the browser.

## Cause

The remote WebDriver returned an error response. The constructor
takes a `reason: Reason` (one of the W3C WebDriver error codes), the
browser-supplied `detail` message, and the browser-side
`browserStacktrace`. The rendered message is `the WebDriver action
failed because $reason: $detail`.

## Variants

### `SN-ta/1.1` ‚Äî `NoSuchElement`

The element addressed by the action was not present in the DOM.

**Example.** `[‚ÜØSN-ta/1.1] the WebDriver action failed because the requested element was not found: ‚Ä¶`

**Resolution.** Verify the locator strategy and target; wait for the
element to appear before acting on it.

### `SN-ta/1.2` ‚Äî `StaleElement`

The element was previously located but has since been removed from
or replaced in the DOM.

**Example.** `[‚ÜØSN-ta/1.2] the WebDriver action failed because the element is no longer attached to the page: ‚Ä¶`

**Resolution.** Re-locate the element after navigation or DOM
mutation before retrying the action.

### `SN-ta/1.3` ‚Äî `NoSuchWindow`

The window or tab handle referenced by the action does not exist.

**Example.** `[‚ÜØSN-ta/1.3] the WebDriver action failed because the requested window was not found: ‚Ä¶`

**Resolution.** Confirm the window handle is current; switch to a
valid window before acting.

### `SN-ta/1.4` ‚Äî `JavascriptError`

A script evaluated by the WebDriver threw an exception.

**Example.** `[‚ÜØSN-ta/1.4] the WebDriver action failed because a JavaScript error occurred while evaluating the request: ‚Ä¶`

**Resolution.** Inspect the script and the supplied `detail` for the
underlying JavaScript exception.

### `SN-ta/1.5` ‚Äî `ScriptTimeout`

An asynchronous script did not complete within the configured
timeout.

**Example.** `[‚ÜØSN-ta/1.5] the WebDriver action failed because the asynchronous script did not complete in time: ‚Ä¶`

**Resolution.** Increase the script timeout, or fix the script so
that it resolves promptly.

### `SN-ta/1.6` ‚Äî `Timeout`

The WebDriver operation timed out before completion.

**Example.** `[‚ÜØSN-ta/1.6] the WebDriver action failed because the operation timed out: ‚Ä¶`

**Resolution.** Increase the relevant timeout, or fix the page so
that the awaited condition can be met.

### `SN-ta/1.7` ‚Äî `UnknownCommand`

The WebDriver does not recognise the command issued.

**Example.** `[‚ÜØSN-ta/1.7] the WebDriver action failed because the WebDriver does not recognise the command: ‚Ä¶`

**Resolution.** Confirm the WebDriver version supports the action;
upgrade the driver if necessary.

### `SN-ta/1.8` ‚Äî `Other`

The WebDriver reported an error code that does not map to one of the
above cases. The original code is captured in the `Other(code)`
parameter.

**Example.** `[‚ÜØSN-ta/1.8] the WebDriver action failed because the WebDriver reported the error code element click intercepted: ‚Ä¶`

**Resolution.** Consult the W3C WebDriver specification for the
reported code.
