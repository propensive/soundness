# `SN-ta/1` — `WebDriverError`

Raised when Tarantula's WebDriver action fails inside the browser.

## Cause

The browser-side WebDriver implementation reported an error. The
constructor records the WebDriver `error` code, the browser-supplied
message (`wdMsg`), and the browser-side stack trace.

## Example message

```
[↯SN-ta/1] the action caused the error stale element reference in the browser, with the message: element is not attached to the page document
```

## Resolution

Use the `error` and `wdMsg` to identify the underlying WebDriver
condition; inspect the browser stack trace for the failing
location.
