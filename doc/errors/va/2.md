# `SN-va/2` — Type is not `Optional`

Raised at compile time by Vacuous's `check` macro when the type
parameter is not an `Optional`.

## Cause

The macro requires the type to include `Unset.type` as a member of
the union. Otherwise the type cannot be treated as `Optional`.

## Example message

```
[↯SN-va/2] the type Int is not an `Optional`
```

## Resolution

Use `Optional[T]` (which is `T | Unset.type`) at the call site, or
introduce the `Unset` case explicitly.
