# `SN-va/1` — `UnsetError`

Raised when an `Optional` value that is `Unset` is read as if it
were defined.

## Cause

Vacuous's strict access methods throw this error rather than
returning a sentinel when the value is unset.

## Example message

```
[↯SN-va/1] the value was not set
```

## Resolution

Either guard the access with `.lay`/`.let`, or supply a default
via `.or`.
