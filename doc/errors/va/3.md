# `SN-va/3` — Type is abstract

Raised at compile time by Vacuous's `concrete` macro when the type
is abstract.

## Cause

The macro provides evidence that a type is concrete (i.e. it is
not an abstract type member or unbounded type parameter). When the
type is abstract, the macro halts.

## Example message

```
[↯SN-va/3] type T is abstract
```

## Resolution

Use a concrete type at the call site, or pass evidence of
concreteness.
