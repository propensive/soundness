# `SN-va/5` — Value is not `Optional` (`mandatable`)

Raised at compile time by Vacuous's `mandatable` macro when the
value's type is not an `Optional`.

## Cause

`Mandatable` is the typeclass that converts an `Optional[T]` to a
mandatory `T` (or fails). The macro requires the type to contain
`Unset.type` in its union.

## Example message

```
[↯SN-va/5] the value is not an `Optional`
```

## Resolution

Use an `Optional[T]` value at the call site.
