# `SN-ur/5` — `OfflineError`

Raised when an Internet connection is required but not available.

## Cause

Urticose throws this error when an `Online`-requiring operation is
attempted in a context that has been declared offline (or when
network access fails outright).

## Example message

```
[↯SN-ur/5] an Internet connection is not available
```

## Resolution

Restore network connectivity, or run the operation in a context
where being offline is acceptable.
