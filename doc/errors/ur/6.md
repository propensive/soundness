# `SN-ur/6` — `PortError`

Raised when a value is not a valid TCP/UDP port number.

## Cause

Urticose constrains port numbers to the range 1–65535. Anything
outside is rejected.

## Example message

```
[↯SN-ur/6] the port is not in the valid range
```

## Resolution

Use a port between 1 and 65535.
