# `SN-oc/2` — `GitRefError`

Raised when a value is not a valid Git reference name.

## Cause

Octogenarian validates reference names against Git's reference-name
rules — no spaces, no `..`, no leading dash, etc. Anything that
fails those rules is rejected.

## Example message

```
[↯SN-oc/2] foo bar is not a valid Git reference
```

## Resolution

Use a name that complies with `git check-ref-format`.
