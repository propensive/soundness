# `SN-sc/1` — `ServerError`

Raised when Scintillate cannot start an HTTP server on the
requested port.

## Cause

The OS refused to bind the listener — typically because the port is
already in use, or it is privileged and the process lacks
permission.

## Example message

```
[↯SN-sc/1] could not start an HTTP server on port 80
```

## Resolution

Choose a different port, free the conflicting one, or run with
sufficient privilege.
