# `SN-cl/1` — `BcodlError`

Raised when Cellulose's BCoDL (binary CoDL) parser encounters
unexpected input.

## Cause

The parser reports an expected token or structure (`expectation`)
along with the byte `position` at which the mismatch was detected.

## Example message

```
[↯SN-cl/1] expected key at position 24
```

## Resolution

Inspect the input around the indicated position; ensure it conforms
to the BCoDL grammar.
