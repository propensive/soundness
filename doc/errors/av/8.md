# `SN-av/8` — Time literal: minutes must be two digits

Raised by Aviation's `validTime` macro when the literal does not
have exactly two minute digits after the decimal point.

## Cause

The macro inspects the source text of the literal and requires the
form `<hour>.<mm>` where `<mm>` is two digits.

## Example message

```
[↯SN-av/8] the time should have exactly two minutes digits
```

## Resolution

Pad the minutes to two digits (e.g. write `9.05` rather than `9.5`).
