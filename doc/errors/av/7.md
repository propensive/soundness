# `SN-av/7` — Time literal: hour value above 12

Raised by Aviation's `validTime` macro when an hour value exceeds
12.

## Cause

The literal time syntax uses 12-hour notation, qualified by `am`/
`pm`. Hours above 12 cannot be encoded.

## Example message

```
[↯SN-av/7] a time cannot have an hour value above 12
```

## Resolution

Use an hour value 0–12; specify `am`/`pm` to disambiguate.
