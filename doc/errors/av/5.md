# `SN-av/5` — Time literal: minute value above 59

Raised by Aviation's `validTime` macro when a time literal's minute
component is 60 or more.

## Cause

The macro decodes literal `Double` values into a `Clockface`. The
fractional part is interpreted as two minute digits; a value of 60
or more is invalid.

## Example message

```
[↯SN-av/5] a time cannot have a minute value above 59
```

## Resolution

Use a minute value 0–59 in the literal.
