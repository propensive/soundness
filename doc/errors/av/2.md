# `SN-av/2` — `TimestampError`

Raised when Aviation cannot parse a timestamp string.

## Cause

Aviation's timestamp parser expects a recognised timestamp format.
Any input it cannot interpret raises this error.

## Example message

```
[↯SN-av/2] the time 2026/01/02 could not be parsed
```

## Resolution

Provide the timestamp in a recognised format (ISO 8601 is the
default); pre-process non-standard inputs before feeding them to
Aviation.
