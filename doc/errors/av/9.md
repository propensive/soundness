# `SN-av/9` — Time literal must be a `Double`

Raised by Aviation's `validTime` macro when the argument is not a
literal `Double`.

## Cause

The macro pattern-matches on `Literal(DoubleConstant(_))`; anything
else (a variable, a method call, an integer literal) cannot be
inspected at compile time.

## Example message

```
[↯SN-av/9] expected a literal double value
```

## Resolution

Pass a `Double` literal directly to the macro.
