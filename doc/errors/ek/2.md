# `SN-ek/2` — `TmuxError`

Raised when Exoskeleton cannot execute the `tmux` binary.

## Cause

Exoskeleton's `Tmux` integration shells out to `tmux`. If the
binary is not on `PATH` (or is not executable), this error is
thrown.

## Example message

```
[↯SN-ek/2] can't execute tmux
```

## Resolution

Install `tmux` and ensure it is on the `PATH` of the process.
