# `SN-ek/1` — `InstallError`

Raised when Exoskeleton cannot install a command-line application.

## Cause

Exoskeleton's installer writes shell-completion and launcher files
into the user's environment. This error reports a failure either at
the discovery stage (working out where to write) or at the I/O stage
(actually writing). The message is `the installation failed because
$reason`.

## Variants

### `SN-ek/1.1` — `Environment`

The installer could not gather enough information about the
environment — for example a required environment variable or shell
configuration file is missing.

**Example.** `[↯SN-ek/1.1] the installation failed because it was not possible to get enough information about the install environment`

**Resolution.** Run the installer in an environment where the
expected variables and config files are available; consult the
documentation for the supported environments.

### `SN-ek/1.2` — `Io`

An I/O error occurred while writing one of the installation files.

**Example.** `[↯SN-ek/1.2] the installation failed because an I/O error occurred when trying to write an installation file`

**Resolution.** Check filesystem permissions and free space at the
target locations.
