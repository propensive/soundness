# `SN-dl/1` — `NumberError`

Raised when Distillate cannot parse a text value as a numeric type.

## Cause

Distillate's `Decodable in Text` instances for numeric primitives
parse the input via the JDK's number parsers. When parsing fails,
this error is thrown with the offending `text` and the
`specializable` (the target numeric type).

## Example message

```
[↯SN-dl/1] abc is not a valid Int
```

## Resolution

Sanitise the input before decoding, or guard against non-numeric
values.
