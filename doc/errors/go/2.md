# `SN-go/2` — `RangeError`

Raised when an index falls outside a permitted range.

## Cause

Gossamer uses `RangeError` for index-style access (string
slicing, array indexing) when the requested index is not in
`[from, to]`.

## Example message

```
[↯SN-go/2] the index 9 is outside the range 0-5
```

## Resolution

Bounds-check before indexing, or use an `Optional`-returning
accessor.
