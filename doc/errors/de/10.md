# `SN-de/10` — Blank line after imports

Raised by the Decorum compiler plugin when there is no blank line
between the last import and the first declaration in the file.

## Cause

A single blank line must separate the import section from the
body.

## Example message

```
[↯SN-de/10] missing blank line between imports and first declaration
```

## Resolution

Add a blank line after the last `import`.
