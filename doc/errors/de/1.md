# `SN-de/1` — Tab characters

Raised by the Decorum compiler plugin when a tab character appears
anywhere in source.

## Cause

Soundness style requires that all indentation and intra-line spacing
use spaces, not tabs.

## Example message

```
[↯SN-de/1] tab character is not permitted; use spaces
```

## Resolution

Configure the editor to insert spaces; replace any existing tabs
with the appropriate number of spaces.
