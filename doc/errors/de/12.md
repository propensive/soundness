# `SN-de/12` — Bracket inner spacing

Raised by the Decorum compiler plugin when there is unwanted space
immediately inside a single-line bracket pair.

## Cause

In single-line constructs, no space is permitted directly after an
opening bracket or directly before a closing bracket.

## Example messages

```
[↯SN-de/12] no space is permitted directly after `(`
```

```
[↯SN-de/12] no space is permitted directly before `)`
```

## Resolution

Remove the space adjacent to the bracket; multi-line forms use a
different rule (see `SN-de/30`).
