# `SN-de/8` — Blank line after `package`

Raised by the Decorum compiler plugin when line 34 (immediately
after the `package` declaration) is not blank.

## Cause

A single blank line must separate the `package` declaration from
whatever follows.

## Example message

```
[↯SN-de/8] line 34 must be a single blank line after `package`
```

## Resolution

Insert a single blank line on line 34.
