# `SN-de/19` — `=>` alignment in case run

Raised by the Decorum compiler plugin when a `case` clause's `=>`
is not aligned with the run of cases it belongs to.

## Cause

Within a contiguous run of `case` clauses, every `=>` should be in
the same column.

## Example message

```
[↯SN-de/19] `=>` should be at column 17 to align with the case run
```

## Resolution

Adjust the spacing before each `=>` so all of them sit in the same
column within the run.
