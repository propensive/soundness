# `SN-de/28` — `object` must precede `class`/`trait`/`enum` of same name

Raised by the Decorum compiler plugin when a companion `object`
appears below the type it accompanies.

## Cause

Soundness style places the companion `object` before the
`class`/`trait`/`enum` it pairs with.

## Example message

```
[↯SN-de/28] object `Foo` must appear before class/trait/enum `Foo`
```

## Resolution

Move the `object` declaration above the type declaration.
