# `SN-de/3` — Indent width must be even

Raised by the Decorum compiler plugin when a non-blank line's
leading spaces are not a multiple of 2.

## Cause

Soundness style indents in steps of two columns.

## Example message

```
[↯SN-de/3] indent width 3 is not a multiple of 2
```

## Resolution

Adjust the indentation to the nearest multiple of 2.
