# `SN-de/6` — Licence-header block comment

Raised by the Decorum compiler plugin when the licence-header block
comment is missing or malformed.

## Cause

Every Soundness source file must open with a `/* … */` licence
header. Line 1 must contain `/*` and line 32 must contain `*/`.

## Example message

```
[↯SN-de/6] line 1 must open the license-header block comment with `/*`
```

```
[↯SN-de/6] line 32 must close the license-header block comment with `*/`
```

## Resolution

Copy the standard 32-line licence header from another module's
source files.
