# `SN-de/5` — More than two consecutive blank lines

Raised by the Decorum compiler plugin when a third (or further)
blank line appears in a run.

## Cause

Soundness style allows at most two consecutive blank lines.

## Example message

```
[↯SN-de/5] more than two consecutive blank lines
```

## Resolution

Remove the surplus blank lines.
