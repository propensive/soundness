# `SN-de/4` — Trailing whitespace

Raised by the Decorum compiler plugin when a line has whitespace
after its last visible character.

## Cause

Soundness style forbids trailing spaces or tabs on any line.

## Example message

```
[↯SN-de/4] line has trailing whitespace
```

## Resolution

Strip the trailing whitespace; many editors can do this on save.
