# `SN-de/2` — Line length

Raised by the Decorum compiler plugin when a line exceeds 100
visible columns.

## Cause

Soundness style limits each line to 100 visible columns
(zero-width and combining characters do not count).

## Example message

```
[↯SN-de/2] line exceeds 100 columns (is 117 columns)
```

## Resolution

Wrap the line; long expressions can usually be broken at function
calls, operators, or after `=`.
