# `SN-de/21` — Blank line after heavy-signature return type

Raised by the Decorum compiler plugin when a heavy-signature
declaration's return-type line is not followed by a blank line
before the body.

## Cause

When a method's signature spans multiple lines and its return type
appears on its own line, the body must be separated from it by a
blank line.

## Example message

```
[↯SN-de/21] a blank line is required between a heavy-signature return type and the body
```

## Resolution

Insert a blank line after the return-type line.
