# `SN-de/30` — Multi-line block bracket spacing

Raised by the Decorum compiler plugin when bracket spacing in a
multi-line block does not follow the documented form.

## Cause

In a multi-line block, the opening bracket must be followed by a
space, the closing bracket must be preceded by a space, and the
closing bracket must not appear alone on its own line.

## Example messages

```
[↯SN-de/30] a space is required after `(` in a multi-line block
```

```
[↯SN-de/30] a space is required before `)` in a multi-line block
```

```
[↯SN-de/30] `)` cannot appear alone on its line; the closing bracket of a multi-line block goes on the same line as the last parameter
```

## Resolution

Adjust the bracket spacing to match the rule, or convert the block
to single-line form (which uses different spacing — see `SN-de/12`).
