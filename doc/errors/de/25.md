# `SN-de/25` — Continuation indents after `=>` and heavy `:`

Raised by the Decorum compiler plugin when a continuation line
following a `=>` or heavy-signature `:` does not have the required
fixed indent.

## Cause

A `=>` continuation line must be followed by exactly two spaces; a
heavy-signature return-type `:` continuation must be followed by
exactly three spaces.

## Example messages

```
[↯SN-de/25] `=>` continuation line must be followed by exactly two spaces
```

```
[↯SN-de/25] heavy-signature return type `:` must be followed by exactly three spaces
```

## Resolution

Adjust the leading whitespace on the continuation line to match.
