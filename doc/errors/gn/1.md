# `SN-gn/1` — `ReferenceError`

Raised by Gnossienne when a reference cannot be resolved.

## Cause

A reference token (typically embedded in markdown or another
structured document) did not point to a known target.

## Example message

```
[↯SN-gn/1] reference foo.md could not be resolved
```

## Resolution

Confirm the target exists and the reference text matches it
exactly (case sensitivity, anchors, etc.).
