# `SN-gn/1` — `ReferenceError`

Raised by Gnossienne when a reference cannot be resolved.

## Cause

A reference token (typically embedded in markdown or another
structured document) could not be resolved to a target. The
constructor takes the offending `reference` text along with a
`reason: Reason`; the rendered message is `the reference $reference
could not be resolved because $reason`.

## Variants

### `SN-gn/1.1` — `NotFound`

No target with the given reference was present in the resolver's
store.

**Example.** `[↯SN-gn/1.1] the reference foo.md could not be resolved because no target with that reference was found in the store`

**Resolution.** Confirm the target exists and the reference text
matches it exactly (case sensitivity, anchors, etc.).
