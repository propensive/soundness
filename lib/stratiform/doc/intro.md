_Stratiform_ is a Scala implementation of TEL — the
[Typed Element Language](https://github.com/propensive/tel) — a
tree-structured, indentation-based data language with schemas, schema
layers, and presentation-preserving mutation. TEL is an evolution of
[CODL](https://github.com/propensive/cellulose) with richer schema
machinery, validators, and a deterministic binary encoding (BinTEL).
